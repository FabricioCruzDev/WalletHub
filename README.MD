💰 WALLTHUB
O WALLTHUB é uma solução de gerenciamento financeiro pessoal projetada para consolidar gastos de débito e crédito em um único lugar. O foco do sistema é oferecer uma visão clara do consumo do usuário, categorizando despesas de forma inteligente e fornecendo alertas críticos para a saúde financeira.

🚀 Funcionalidades Principal
Consolidação de Gastos: Unificação de transações via débito e crédito.

Categorização: Registro e organização automática/manual de despesas por categoria.

Visão de Saldo: Exibição de saldo total e parcial.

Inteligência Financeira: Geração de estimativas de gastos futuros com base no histórico.

Sistema de Alertas: Notificações sobre vencimentos de faturas e contas.

🛠️ Stack Tecnológica
Linguagem: Python 3.14.3

Framework Web: Flask

Banco de Dados & Auth: Supabase (PostgreSQL + Realtime)

Gerenciamento de Dependências: pip

📥 Instalação e Configuração
Pré-requisitos
Python 3.14.3 instalado.

Conta ativa no Supabase.
